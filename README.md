# Mineradio 二次开发版

本项目基于 [XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio) 进行二次开发。原项目是一款 Windows Electron 桌面沉浸式音乐播放器，融合天气电台、搜索播放、歌词舞台、粒子视觉、3D 歌单架和本地桌面体验。

在原项目基础上，本版本主要补充了酷狗音乐相关能力，并对部分运行路径做了小范围优化。由于我在个人电脑上打开原项目软件时会遇到轻微卡顿，所以顺手对代码做了一些保守的流畅度优化；不同电脑环境下效果可能不同，流畅度提升不作绝对保证。

后面可能会对这个项目做进一步的优化，并添加更多功能，不知道为什么总感觉软件流畅度还是有点问题，之前在网上看到有人说想要接入汽水音乐，这两天可能会试试，Spotify（声破天）的话因为Spotify自己的原因接入好像还挺麻烦的，不知道XxHuberrr大佬会不会做

## 二次开发内容

- 增加酷狗音乐登录方式，支持通过桌面端网页登录窗口同步酷狗音乐会话。
- 支持手动导入酷狗音乐 Cookie，用于补充登录状态。
- 支持读取并导入酷狗音乐中的个人歌单，在软件内与网易云音乐、QQ 音乐歌单一起展示和播放。
- 保留原项目的网易云音乐账号、QQ 音乐搜索/登录、天气电台、歌词舞台、粒子视觉和 3D 歌单架等核心体验。
- 对部分代码做了不改变功能的优化，包括减少高频调试日志、异步化部分文件读写、降低重复刷新和重复 DOM 查询带来的开销。

## 核心特性

- Open-Meteo 天气电台，根据当前位置、城市和天气 mood 生成更合适的播放队列
- 首页包含天气电台、每日推荐、私人电台、继续听、听歌画像和我的歌单入口
- Wallpaper 银河首页背景，未播放状态保持干净的星河氛围
- 播放后切换到 Emily / 默认播放态视觉，歌词舞台与粒子舞台同步工作
- 基于节奏的电影镜头视觉系统
- 面向长播客和 DJ 曲目的专属视觉模式
- 歌词舞台、自定义歌词、歌词位置与视觉控制
- 自定义专辑封面上传与裁剪
- 右键唤起 3D 歌单架，支持歌单队列浏览
- 网易云音乐账号、搜索、歌单、播客等体验接入
- QQ 音乐搜索、登录态与音源补充接入
- 酷狗音乐登录、会话同步、个人歌单读取与歌单歌曲加载
- GitHub Releases 更新检测与下载入口
- 首次启动内置「默认测试」视觉用户存档，软件内默认视觉参数与该存档一致

## 开发运行

```bash
npm install
npm start
```

桌面版入口由 Electron 主进程加载本地服务。需要打包 Windows 安装包时可以运行：

```bash
npm run build:win
```

打包产物默认位于 `dist/`，不建议把 `dist/`、`node_modules/` 或本机登录数据提交到 GitHub。

## 上传 GitHub 前注意

上传源码时请只提交项目代码、配置、文档和必要资源。以下内容不应提交：

```text
node_modules/
dist/
updates/
.cookie
.qq-cookie
.kugou-cookie
*.log
```

登录 Cookie、搜索历史、自定义封面、自定义歌词、节奏分析缓存等数据只应保存在本机用户数据目录或浏览器本地存储中，不应提交到公开仓库。

## 第三方音乐平台说明

本项目不是网易云音乐、QQ 音乐、酷狗音乐或腾讯音乐娱乐集团的官方客户端，也不隶属于任何音乐平台。

项目中的第三方平台接入仅用于个人学习、本地客户端体验和用户自有账号的播放辅助。请遵守对应平台的用户协议、版权规则和会员权益规则。项目不会提供绕过付费、绕过会员、破解音质或重新分发音乐内容的能力。

酷狗音乐相关功能依赖用户自己的账号登录状态和网页接口可用性。若平台接口、网页登录流程或账号状态发生变化，相关能力可能需要重新适配。
